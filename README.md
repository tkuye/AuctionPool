# Welcome to AuctionPool! 
## This is a simple way for you to make money fast and easily. 
## How it works:
### You start by whether you want to be an auctioneer or a buyer. 
**If you want to be an auctioneer:**
- You must have at least one item worth selling to begin. 
- You must then create a pool with the item you are selling. 
- After creating the pool people will then join and there they can bid for the products you are selling. 


**If you want to be a buyer:**
- You must join a pool and begin the bidding!
- After you have an item you can choose to either continue bidding or leave the pool. 
- If you desire you can immediately create another pool with the items you bidded for to resell them at a potentially higher price. 

### *Please have fun and enjoy using AuctionPool!*
